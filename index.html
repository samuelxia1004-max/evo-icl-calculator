方向沟径　STSeff = 1 / √(cos²θ / STS-H² + sin²θ / STS-V²)
WTW 回退（探索性）　STS-H ≈ 2.627 + 0.7888×WTW；STS-V ≈ 2.780 + 0.8145×WTW （队列内 MAE 约 0.23 / 0.28 mm）

解释重点。WTW 与拱高有明显单变量关联，但它同时影响临床 ICL 尺寸选择，又只是对内部睫状沟大小的外部代理；在本队列中，当 ACD、计划尺寸、LT 和方向校正 STS 已知后，加入 WTW 没有改善交叉验证性能。反过来，STS 与植入方向联合使用可提供额外信息。模型输出属于群体层面的研究估计；点预测、经验区间和近似阈值概率都不能被理解为安全保证。
